# Product Overview

Nani風AI翻訳デスクトップアプリは、macOSユーザー向けのショートカットキー翻訳ツールです。任意のアプリケーションで選択したテキストを、グローバルショートカット一つで即座に翻訳し、ポップアップ表示します。

## Core Capabilities

1. **グローバルショートカット翻訳**: システム全体で動作するホットキーで、どのアプリからでも翻訳を起動
2. **言語自動検出**: 入力テキストの言語を自動判定し、適切な翻訳方向を決定
3. **デュアルLLM対応**: Ollama（ローカル）とClaude API（リモート）を切り替え可能
4. **軽量ポップアップUI**: 作業を妨げない小さなウィンドウで翻訳結果を表示

## Target Use Cases

- **開発者**: 英語ドキュメントやStack Overflowの回答を素早く日本語で理解
- **翻訳作業**: テキストエディタやブラウザで選択したテキストを即座に翻訳
- **オフライン利用**: インターネット接続なしでもOllamaによるローカル翻訳が可能
- **プライバシー重視**: 機密情報をローカルLLMで翻訳し、外部送信を回避

## Value Proposition

- **ワンアクション翻訳**: Cmd+Shift+T一つで選択テキストを翻訳、コピー&ペーストの手間を削減
- **プライバシー選択**: ローカルLLM（Ollama）と高精度リモートLLM（Claude）を用途に応じて選択
- **軽量設計**: Tauri v2による10MB以下のアプリサイズ、低メモリフットプリント
- **個人利用最適化**: シンプルな設定、直感的なUI
